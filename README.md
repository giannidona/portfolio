## IMPLEMENTATION NOTES

- **Hero CTA** should link to your contact form or Calendly — not email or Instagram
- **Status badge** ("Available") should be easy to toggle when you're fully booked
- **About section** — remove all mentions of "studying" or "pursuing a degree" from the current site. It undermines confidence.
- **Projects** — each project card should have: title, 1-sentence description, stack used, and a live link. Screenshots alone aren't enough.
- **Services section** is new — your current site doesn't have one. Adding it makes it clear what you sell.
- Consider adding a **testimonial** if you have even one happy client. One real quote beats a perfect design.
